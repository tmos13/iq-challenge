# IQ Challenge

**A TMOS13 pack for cognitive profiling.**

Verbal reasoning. Numerical reasoning. Pattern recognition.
Spatial logic. Four sections, adaptive difficulty, scored profile.

Not a clinical IQ test — a rigorous cognitive challenge that
produces a meaningful profile of where your thinking is strongest.

### Four sections

- **Verbal Reasoning** — analogies, vocabulary in context, logical inference from text
- **Numerical Reasoning** — number series, arithmetic reasoning, data interpretation
- **Pattern Recognition** — sequence completion, matrix reasoning, odd-one-out
- **Spatial Reasoning** — mental rotation, folding, 2D/3D transformation

### The deliverable

`cognitive_profile` — section scores, overall estimate, strongest and
weakest domains, and a cognitive signature: what kind of thinker you are.
